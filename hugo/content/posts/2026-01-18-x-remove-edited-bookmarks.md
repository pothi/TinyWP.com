I was browsing through my bookmarks on X (formerly Twitter) using the mobile app. I was removing a few of the bookmarks that I found useless in the future. However, on a particular tweet, I didn't see the option to remove the bookmark. It was marked as there is a newer version available. When I visit the latest version, I can't see the bookmark there either. Decided to look into this later.

After a while, I opened my desktop and I woke-up Grok to find a solution. Since, Grok is closed tied to X, I thought I would get the right answer almost immediately. I was wrong. Like [the last post on how to backup a new external drive on macOS](https://www.tinywp.com/posts/backup-new-external-drive/), Grok threw an incomplete answer. Anyway, I wanted to follow the workarounds mentioned in Grok and visited the bookmark section on Twitter. I can see the option to remove the bookmark of the same old edited version of the tweet. Sigh! It was never mentioned by Grok. All I had to do is to use the desktop/laptop and browse X via a browser.
